35 Comandos Básicos do Python

  • print() - Exibe informações na tela.
  • input() - Recebe entrada do usuário.
  • int(), float(), str() - Converte tipos de dados.
  • len() - Retorna o tamanho de listas, strings, etc.
  • type() - Mostra o tipo do dado.
  • if, elif, else - Condicionais.
  • for - Loop que percorre itens.
  • while - Loop baseado em condição.
  • break - Encerra um loop.
  • continue - Pula para a próxima iteração do loop.
  • def - Define uma função.
  • return - Retorna um valor de uma função.
  • import - Importa módulos inteiros.
  • from ... import ... - Importa parte específica de um módulo.
  • as - Cria um apelido para um módulo.
  • list() - Cria uma lista.
  • tuple() - Cria uma tupla (imutável).
  • dict() - Cria um dicionário.
  • set() - Cria um conjunto de itens únicos.
  • append() - Adiciona item a uma lista.
  • remove() - Remove item da lista.
  • pop() - Remove item de uma lista e o retorna.
  • sort() - Ordena uma lista.
  • reverse() - Inverte a ordem de uma lista.
  • open() - Abre um arquivo.
  • read(), readline(), readlines() - Lê o conteúdo de arquivos.
  • write() - Escreve em arquivos.
  • with open() as - Abre arquivos de maneira segura.
  • try, except - Tratamento de erros.
  • finally - Executa sempre após try/except.
  • class - Define uma classe.
  • self - Representa a instância atual de uma classe.
  • init() - Método construtor de uma classe.
  • super() - Chama métodos da classe pai.
  • lambda - Cria funções anônimas.
